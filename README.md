# Arifa — Portfolio
A personal website built with HTML and CSS, optimized for GitHub Pages hosting.

## Features

- Responsive layout with a modern glassmorphism aesthetic
- **Light/Dark mode toggle** in the navigation bar for theme switching
- Sections for hero, about, skills, education, projects, and contact
- SEO meta tags, Open Graph, and JSON‑LD structured data for better visibility
- Contact form powered by Formspree
- Mobile-friendly navigation and accessibility improvements
